# Mechanical Engineering Portfolio

Static one-page portfolio site. Deployed on Vercel.

## Local setup
1. Place all 8 project images in this folder root (same names referenced in index.html):
   - FORMULA-IUT-IMPACT-SIMULATION.jpg
   - FORMULA-IUT-IRL.jpg
   - IMPACT-ATTENUATOR-FULL-WINDOW-SIMULATION.jpg
   - IMPACT-ATTENUATOR-SIMULATION.jpg
   - STIRLING-ENGINE-CAD.jpg
   - STIRLING-ENGINE-IRL.jpg
   - SWISS-ARMY-KNIFE-MOULD.jpg
   - UAS-CHALLENGE-AIRFLOW-SIMULATION.jpg
2. Open index.html directly in a browser to preview.

## Deploy
Push to GitHub, then import the repo in Vercel (Framework Preset: Other / Static).
